Controlling IoT devices with Tobii glasses

The current project idea is to be able to controll certain connected devices and turn them on and off just with the use of our eyes. To control a specific device the user should be able to just look at that device (or maybe a sticker next to it) and blink to turn the device on or off. For the demonstration we plan to use the Phillips Hue system or some of the Z-wave compatible devices.

 

The project idea is not yet finalized and there are techonogical aspects that we still need to figure out how to solve. This page will be updated as we progress through the course.
