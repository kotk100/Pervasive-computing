Controlling IoT Devices with Tobii Glasses

IoT devices and smart homes are becoming more and more popular. With the addition of so many devices in our home we also need a good way to interact with them. While there are many human-computer interaction (HCI) models, like the more and more popular voice assistants, not every method is appropriate for controlling a pervasive environment. One of the possible interaction models is with the use of gaze and eye tracking of a person. Our project will focus on researching and implementing a prototype system which will allow us to control devices like the Philips Hue system simply with our eyes and possibly complemented with another input method like the before mentioned voice commands.

The project idea is to be able to controll certain connected devices and turn them on and off among other things, just with the use of our eyes. To control a specific device the user should be able to just look at that device and blink to interact with that device. For the demonstration we will use the Philips Hue system.


Status


The project is currently fully compatible with the tobii glasses and Philips Hue lights and can support control for up to three lights at the same time.

The controlls that are fully supported are as follows:

• Controllong the state of up to three hue lights with Tobii glasses

• Change individual color for all Hue lights connected

• Adjusting brightness for all Hue lights connected

About


The eye-tracking glasses we use are the Tobii Pro 2 glasses, which are used to determine where the user is looking at and detect users blinks. This information is used in the main control unit to determine which light the user is looking at and to control things like the brightness, color and state of the lights. The system currently only supports the control of lights which are part of the Phillips hue system connected to the hub which is under control of the control unit, which in our case is just a personal computer, but could be anything from a raspberry pi to a cloud service. For the system to work it is required that the Philips Hue system is connected to the same sub-network as the control unit and the Tobii gaze tracking glasses have to be connected directly to the control unit via a wireless connection.


This project is part of the course pervasive computing being conducted by a group of three students in the LP2 quarter of 2018 at LTU (Lulea Technical University), Sweden.
